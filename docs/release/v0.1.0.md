## Installation and verification

```zsh
pnpm install --frozen-lockfile
pnpm test
npm test --prefix demo/password-reset-workspace
node packages/cli/dist/index.js verify fixtures/judge-passport/passport.json fixtures/judge-passport/artifacts --json
```

The local capture route is tested on macOS with desktop-bundled `codex-cli 0.145.0-alpha.18`. The hosted verifier requires no account, OpenAI Application Programming Interface key, or local code execution.

## Known limitations

- Durable audit delivery requires a production queue or transactional outbox.
- Token atomicity is demonstrated within one process; distributed workers and restarts require persistent shared infrastructure.
- The signature proves integrity under the corresponding key. It does not prove correctness, signer identity, certified security, legal compliance, or trusted time.
- Live Codex approval exchange remains unverified; the public capture correctly records approval coverage as not observed.
