## Description

Code generation creates output. Serious engineering also needs evidence that can survive inspection.

Flight Recorder is a verifiable engineering passport for software built with Codex. It captures observable engineering activity, links it to final committed artifacts, adds evidence-referenced GPT-5.6 assurance review, records human decisions, and seals the canonical record with an Ed25519 signature.

Try the public, no-login demonstration:
https://flight-recorder-web.vercel.app/

Source code and Apache License 2.0 release:
https://github.com/Procureallyai/flight-recorder/releases/tag/v0.1.0

The demonstration shows:

- a sanitised genuine Codex session;
- four GPT-5.6 specialist reviews and one synthesis review;
- a signed passport with independently recomputed artifact hashes, event chain, Merkle root, and signature;
- four disclosed demonstration-scope warnings;
- an in-memory covered-artifact change that turns verification from valid to invalid; and
- restoration of the original artifact and successful reverification.

The integrity claim is deliberately narrow: the covered evidence has not changed since sealing by the corresponding key holder. Flight Recorder does not claim correctness, certified security, verified signer identity, legal compliance, or trusted time.

This video uses synthetic narration. No private reasoning, Application Programming Interface key, private signing key, or private `/feedback` Session Identifier appears in the recording.
